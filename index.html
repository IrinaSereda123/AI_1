📖 Часть 1. Что такое API?

API (Application Programming Interface) — это способ, с помощью которого разные программы могут общаться друг с другом. Представь, что ты в ресторане: ты (твоя программа) делаешь заказ официанту (API), а он приносит его с кухни (сервер).

🔍 Как это работает в нашей задаче:
  • Твоя страница отправляет вопрос к Groq API.
  • Groq API передаёт вопрос нейросети (модели Llama 3.3).
  • Нейросеть генерирует ответ и возвращает его через API.
  • Твоя страница получает ответ и показывает его пользователю.

🌐 Что такое Groq?

Groq — это сервис, который предоставляет бесплатный доступ к мощным языковым моделям (как ChatGPT). У них есть API, к которому мы можем обращаться из JavaScript.

Для работы нам понадобится API-ключ — это как пароль, который даёт доступ к сервису.

🚀 Шаг 3. JavaScript — подключение к API

✏️ Твоя задача:

Создай файл script.js и скопируй этот код. Вместо ВСТАВЬ_СВОЙ_КЛЮЧ нужно будет вставить свой API-ключ, который ты получишь на сайте Groq.

🔑 Как получить API-ключ (бесплатно):
1. Перейди на https://console.groq.com.
2. Зарегистрируйся (можно через Google или GitHub).
3. Перейди в раздел «API Keys» → «Create API Key».
4. Скопируй ключ и вставь его в код вместо ВСТАВЬ_СВОЙ_КЛЮЧ. 
// --- Ждём, пока загрузится страница ---
document.addEventListener('DOMContentLoaded', function() {
    // --- Находим элементы на странице ---
    const button = document.getElementById('askBtn');
    const questionInput = document.getElementById('questionInput');
    const answerDiv = document.getElementById('answerResult');

    // --- ОЧИСТКА ПЛЕЙСХОЛДЕРА ПРИ КЛИКЕ ---
    const defaultQuestion = "Что такое искусственный интеллект?";
    
    questionInput.addEventListener('focus', function() {
        if (questionInput.value === defaultQuestion) {
            questionInput.value = '';
            questionInput.style.color = '#333';
        }
    });
    
    questionInput.addEventListener('blur', function() {
        if (questionInput.value === '') {
            questionInput.value = defaultQuestion;
            questionInput.style.color = '#aaa';
        }
    });

    // --- ВСТАВЬ СЮДА СВОЙ API-КЛЮЧ (получи на groq.com) ---
    const apiKey = 'ВСТАВЬ_СВОЙ_КЛЮЧ_СЮДА';

    // --- Системный промпт (как AI должен себя вести) ---
    const systemPrompt = `Ты дружелюбный AI-помощник для детей. 
                          Отвечай кратко (2-3 предложения), понятно и с юмором. 
                          Используй простые слова. Если вопрос сложный, объясняй на примерах.`;

    // --- Функция для отправки вопроса ---
    async function askAI() {
        let question = questionInput.value.trim();

        if (question === defaultQuestion || question === '') {
            answerDiv.innerHTML = '<p class="hint">Пожалуйста, введи свой вопрос!</p>';
            return;
        }

        answerDiv.innerHTML = '<p class="loading">AI думает</p>';

        try {
            const response = await fetch('https://api.groq.com/openai/v1/chat/completions', {
                method: 'POST',
                headers: {
                    'Authorization': `Bearer ${apiKey}`,
                    'Content-Type': 'application/json'
                },
                body: JSON.stringify({
                    model: 'llama-3.3-70b-versatile',
                    messages: [
                        { role: 'system', content: systemPrompt },
                        { role: 'user', content: question }
                    ],
                    temperature: 0.7,
                    max_tokens: 150
                })
            });

            if (!response.ok) {
                const errorData = await response.json();
                throw new Error(errorData.error?.message || `Ошибка ${response.status}`);
            }

            const data = await response.json();
            const answer = data.choices[0].message.content;

            answerDiv.innerHTML = `
                <div class="answer-text">
                    <strong>🤖 AI:</strong> ${answer}
                </div>
                <small style="color: #999; margin-top: 10px; display: block;">
                    Ответ сгенерирован моделью Llama 3.3
                </small>
            `;

        } catch (error) {
            console.error('Ошибка:', error);
            answerDiv.innerHTML = `
                <p class="error">❌ Ошибка: ${error.message}</p>
                <p style="color: #666; font-size: 14px;">
                    Проверь API-ключ или подключение к интернету
                </p>
            `;
        }
    }

    button.addEventListener('click', askAI);
    questionInput.addEventListener('keypress', function(e) {
        if (e.key === 'Enter') askAI();
    });
});
📖 Что нужно знать для понимания кода:
• async/await — специальный синтаксис для работы с запросами.
• fetch() — функция для отправки HTTP-запросов.
• try/catch — ловит ошибки, чтобы программа не падала.
• Bearer ${apiKey} — способ передачи ключа доступа к API.

📚 Шпаргалка по API

КомандаЧто делает
fetch(url)Отправляет запрос к серверу
async/await Позволяет дождаться ответа
response.json() Превращает ответ в объект
try/catch Ловит ошибки
Bearer KEY Передаёт API-ключ
temperature Творчество AI (0-1)
max_tokens Максимальная длина ответа
💡 Частые ошибки:
  • Ошибка 401 — неправильный API-ключ.
  • Ошибка 429 — слишком много запросов, подожди.
  • Нет ответа — проверь консоль браузера (F12).
